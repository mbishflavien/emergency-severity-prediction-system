# Emergency Incident Severity Prediction System

A modern, AI-powered command center dashboard for monitoring and predicting emergency incident severity.

## Features
- **Real-time Overview**: Live map and KPI metrics.
- **AI Prediction**: Probability-based severity assessment.
- **Analytics**: Historical data visualization.
- **Dark Mode UI**: Optimized for 24/7 potentially low-light command center environments.

## Setup & Run

1.  **Install Requirements**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Start the App**
    ```bash
    python -m streamlit run app.py
    ```

3.  **Login**
    -   The system includes a mock login screen.
    -   You can enter any username/password to access the dashboard unless configured otherwise in `app.py`.

## Technology Stack
- **Frontend/Backend**: Streamlit (Python)
- **Visualization**: Plotly Express
- **Data Handling**: Pandas, NumPy
